Sambungkan Peta Kuliner Malang ke Database Online

Halaman ini tugasnya membuat file firebase-config.js secara otomatis dari konfigurasi Firebase yang kamu salin. Kamu tidak perlu mengedit kode manual.

← Kembali ke aplikasi

Yang tetap harus kamu lakukan di akun Google

  1. Buka Firebase Console, buat project baru, misalnya malang-food-map.
  2. Di Project Overview, tekan ikon Web </>, beri nama app, lalu Register app.
  3. Salin bagian konfigurasi yang bentuknya seperti const firebaseConfig = { ... }.
  4. Tempel konfigurasi itu di kotak bawah, lalu klik Buat file firebase-config.js.
  5. Upload file hasil download untuk mengganti firebase-config.js lama di GitHub.

Kenapa tidak bisa 100% otomatis?
Karena pembuatan project Firebase, aktivasi database dan storage harus dilakukan dari akun Google pemilik aplikasi. Aku tidak bisa mengambil alih izin akunmu dari sini.

Checklist Firebase setelah file config dibuat

  1. Firestore Database → Create database.
  2. Storage → Get started / mulai aktifkan Storage.
  3. Firestore Database → Rules → tempel isi file firestore.rules → Publish.
  4. Storage → Rules → tempel isi file storage.rules → Publish.
  5. Refresh halaman GitHub Pages kamu, lalu coba isi nama + email dan tambah titik.
